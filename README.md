# Build instructions
#### Windows
Prerequisites:
- [Microsoft Visual Studio Installer](https://visualstudio.microsoft.com/downloads/)
- Microsoft Visual Studio 2017 and up
- Visual Studio workload: Desktop Development with C++
- Optional feature: MSVC Build Tools for x64/x86 (Latest)

---
Steps:
1. Open Developer Command Prompt for [Visual Studio]
2. Navigate to source directory, e.g. `~\Downloads\src-main`
3. Execute `.\scripts\build` or `.\scripts\run`

---
Steps (with editor, e.g. VS Code):
4. Open Developer Command Prompt for [Visual Studio]
5. Navigate to source location
6. Open editor there, e.g. `code .`
7. Open editor's powershell environment
8. Run `.\scripts\build` or `.\scripts\run` while in source directory
